How to make a github page ?

Step 1 : make a repository with your username as the repository name. Add 'github.io' at the end of the name and make it public.

Step 2 : Open Git Bash and go to the destined folder for the repository.  Then clone the repository to the specified folder using 'git clone [URL of your repository on github]'

Step 3 : go into the repository file and add a new file with the title 'index' and the extension 'html' by using the 'echo'  command. You could insert a simple text like
"Hello World" for the first commit.

Step 4 : To publish the site, push it into your github repository by using 'git add -all', commit with
'git commit -m [short description]', and then push it into the mainrepository by using the
'git push -u origin main' command.

Step 5 : To check the website you've made, go to the repository settings and then click the pages sub settings where you will find the URL of your published site. To edit it, you could use
an IDE or website editing software.
